Reproducing the reporting format of Cheng et al. 2022, Table 1 & Figure 4 — run on this project's own data, not the paper's dataset

These numbers are not comparable to the paper's Table 1. The paper trains on the Interspeech 2020 DNS Challenge dataset — 60,000 training utterances from 500 hours of speech. This run used the same code path (fixed to actually execute), the same student architecture (231K params, matching the paper's reported 0.23M), and real WB-PESQ/STOI — but trained on only 4 training / 1 validation example clips for 20 epochs. Treat this as a working demonstration of the pipeline, not a scientific comparison of the distillation methods.
